# flatex-portfolio-scraper
Automatisierter Scraper für das Flatex-Online-Depot. Liest Depotpositionen und Gesamtwerte per Playwright aus der Weboberfläche aus und speichert sie zeitgestempelt in MySQL und InfluxDB. Geeignet für Linux/Docker und regelmäßige, automatisierte Ausführung (z. B. per Cron).
